## 📄 Sobre o Projeto

Este repositório contém um script em Python desenvolvido para gerenciar orçamentos de uma fábrica de camisetas. O programa é modularizado, utilizando funções para capturar entradas do usuário, processar dados com validação de erros e calcular o valor final de pedidos com descontos progressivos e opções de frete.

Esta é uma versão aprimorada com foco em **boas práticas de programação**, **modularização** e **correção de lógica financeira**.

## 🚀 Funcionalidades

- **Seleção de Modelos:** Interface interativa para escolher entre Manga Curta/Longa, com ou sem estampa.
- **Controle de Quantidade:** Aceita pedidos de 1 até 20.000 unidades, com validação de tipo de dados.
- **Descontos Progressivos:** Aplica descontos automáticos baseados no volume do pedido sobre o valor monetário total.
- **Opções de Frete:** Inclui valores para Transportadora, Sedex ou Retirada na Fábrica.
- **Resumo Detalhado:** Exibe um recibo claro com o valor unitário, quantidade, desconto aplicado em %, valor do frete e o total a pagar.
